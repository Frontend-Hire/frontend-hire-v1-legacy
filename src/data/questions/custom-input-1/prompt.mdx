# Custom Input - 1: Dynamic Time Input

> When a minute is more than just a minute.

## Objectives
- Implement a React component called 'TimeInput.'
- The component should allow users to input number calculations (addition, subtraction, multiplication, division).
- The component should automatically convert and display the result in hours if it exceeds 60 minutes.

## Functional Requirements
1. The component should support basic arithmetic operations on time values: addition, subtraction, multiplication, and division.
2. The result should be displayed in minutes if under 60 or converted to hours if 60 or more.
