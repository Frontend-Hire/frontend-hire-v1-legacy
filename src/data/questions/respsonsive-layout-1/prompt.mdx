# Responsive Layout - 1

> Why did the HTML and CSS go to therapy? Because they couldn't handle their relationship issues without a proper layout!

## Requirements

- A simple layout with fixed headers, left and right asides, basically they should stay in position with scroll.
- Left and Right asides should hide for small viewports.
- The footer should be to the bottom end of the page.

### Instructions

- Write your HTML markup in the **index.html** file. Though for this question, there is no need to update it but feel free to make changes.
- Write your CSS in the **styles.css** file.
- Your solution should use classic CSS. However, if you prefer, you can opt for TailwindCSS.

#### Enabling TailwindCSS (Optional)

If you choose to use TailwindCSS, include this script in the head of your **index.html** file:

```
<script src="https://cdn.tailwindcss.com"></script>
```
