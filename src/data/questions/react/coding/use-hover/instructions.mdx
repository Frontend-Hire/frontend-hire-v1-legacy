# Local setup for the React coding questions

## Instructions

## Setup

1. Clone the repository:

```bash
git clone git@github.com:Frontend-Hire/useHover-app-react-tdd-typescript.git
```

2. Navigate to the project directory:

```bash
cd useHover-app-react-tdd-typescript
```

3. Install the dependencies:

```bash
pnpm install
```

## Running the tests

To run the tests for the React coding questions, you can use the following command:

```bash
pnpm test
```

This command will run the tests for all the React coding questions.
