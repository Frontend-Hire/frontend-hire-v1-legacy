# Local Instructions

In case of any issues, contact us on [Discord](https://discord.gg/DWAVqksVtx)

## First Time Setup

If you do not have pnpm installed, run the following command:

```sh
npm install -g pnpm
```

For Mac/Linux:

```sh
mkdir dsa-js
cd dsa-js
pnpm init dsa-js
touch index.js
```

For Windows:

```sh
mkdir dsa-js
cd dsa-js
pnpm init -y
echo. > index.js
```

## Install dependencies

```sh
pnpm install
```
