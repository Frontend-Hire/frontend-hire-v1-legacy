# Can you center a div?

> This is a classic joke in the CSS community.

All you gotta do for this question is **center (horizontally and vertically) the div**.

```
<div>Center Me</div>
```

#### Note

- Though this question expects you to write classic CSS a.k.a in the **styles.css** file, you can use _TailwindCSS_ if you want to.
- For enabling _TailwindCSS_, just add the below snippet to the **head** of the **index.html** file.

```
<script src="https://cdn.tailwindcss.com"></script>
```
