## Frontend Hire Tips

- Do <SpecialText>not</SpecialText> jump directly into React without learning foundational JavaScript!
- React at <SpecialText>its core is simple</SpecialText>, and most of the complexity comes from the ecosystem of React.
- If your <SpecialText>foundations are strong</SpecialText>, you can easily be comfortable in the React ecosystem.
- Learn the different <SpecialText>component design patterns</SpecialText> as quickly as possible.
- Have <SpecialText isLink>[this roadmap from roadmap.sh](https://roadmap.sh/react)</SpecialText> in your bookmarks and refer to it often.
